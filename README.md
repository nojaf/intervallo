# Intervallo

A fun side-project to learn about Music Theory and how it fits on the guitar.

## Stack

- TypeScript (checked with [tsgo](https://github.com/nicolo-ribaudo/tc39-proposal-type-annotations))
- React 19
- Vite 8
- Tailwind CSS 4 + DaisyUI 5
- [oxlint](https://oxc.rs/docs/guide/usage/linter) + custom rules
- [oxfmt](https://oxc.rs/docs/guide/usage/formatter)

## Development

```sh
bun run dev
```

## Quality checks

```sh
bun run ci
```
